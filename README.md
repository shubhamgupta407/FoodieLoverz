## ⚡ What's Inside

### 🧑‍💻 Customer App
| Feature | Details |
|--------|---------|
| 🏪 Multi-outlet browsing | Location selector, outlet cards, ratings |
| 🍔 Menu & Search | Category filters, food cards, item customization |
| 🛒 Cart | Quantity control, delivery fee, coupon codes |
| 📍 Checkout | Saved addresses, no-contact delivery option |
| 🗺️ Live Order Tracking | Real-time map (Leaflet.js), delivery partner info, ETA |
| ✅ Payment Flow | Confirmation screen, order ID generation |

### 🔧 Admin Panel
| Feature | Details |
|--------|---------|
| 📊 Dashboard | Orders, revenue, active users, 7-day revenue trend chart |
| 🍽️ Menu Management | Add, edit, categorize items |
| 📦 Order Management | View and update live orders |
| 🎟️ Coupon System | Create and manage discount codes |
